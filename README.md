## 🚀 使用方式

1. 克隆或下载本仓库到本地。
2. 用浏览器打开对应课程的 `index.html`（如 [`OS/index.html`](OS/index.html)）。
3. 在入口页点击卡片即可跳转到速查表、各章知识清单 / 章节测试或课后作业。

> HTML 资料为纯静态页面，无需联网或安装依赖，直接双击即可打开。

## 🧩 扩展约定

新增内容时遵循以下约定，入口页可自动识别：

- **新增章节**：在 `Knowledge List/` 与 `Chapter Test/` 下按命名规则添加文件
  —— `序号-章节名-知识清单.html` / `序号-章节名-章节测试.html`，
  并在对应 `index.html` 的 `chapters` 数组中登记。
- **新增科目**：在仓库根目录新建科目文件夹（如 `DataStructure/`），
  内含独立的 `index.html`，并在本 README 的「课程目录」表中追加一行。
